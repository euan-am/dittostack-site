About

Dittostack is an application that makes it easy to re-use items you have copied. You can access your copied items by launching Dittostack from any application with a keyboard shortcut (cmd + shift + v). Browse your copy history and choose an item to paste.

Search

Use Dittostack to search your copy history and select items you have previously copied. The search results are displayed in the list on the left and the selected item is displayed as a preview on the right. Once an item has been selected, simply press the enter key to paste it.

Files and Folders

Files and Folders that you have copied will also be displayed in Dittostack. A preview of the selected File or Folder will be displayed in the preview panel on the right.

Works with Multiple Screens

Dittostack will always appear on the screen you are working on.

Security

Items copied from applications that contain sensitive data, such as password managers, can be ignored. These applications can be configured in the preferences.
